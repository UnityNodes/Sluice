Sluice, Predicate JSON Schemas

Drop these URLs into VS Code's json.schemas setting, JetBrains' JSON Schema Mappings, or reference them inline with a $schema key to get inline autocomplete, validation, and hover docs while writing predicates.

predicate-v1.json

The current stable predicate format, AND-of-conditions with 12 operators. View JSON

{
  "$schema": "https://sluice.unitynodes.com/schema/predicate-v1.json",
  "and": [
    { "field": "amount", "op": "gte", "value": "5000000000000" }
  ]
}

VS Code (workspace settings.json)

{
  "json.schemas": [{
    "fileMatch": ["**/predicate*.json", "**/sluice*.json"],
    "url": "https://sluice.unitynodes.com/schema/predicate-v1.json"
  }]
}

JetBrains IDEs

Preferences → Languages & Frameworks → Schemas and DTDs → JSON Schema Mappings → + → paste the URL above and target predicate*.json.

Validate from CLI

# with ajv-cli
npx ajv-cli@5 validate \
  -s https://sluice.unitynodes.com/schema/predicate-v1.json \
  -d ./my-predicate.json
